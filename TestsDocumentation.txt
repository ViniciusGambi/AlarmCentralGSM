DOCUMENTAÇÃO DE TESTE
__________________
1. É CAPAZ DE LIGAR E DESLIGAR O ALARME
-APERTA O BOTÃO

DEVE-SE OUVIR UM BEEP PARA LIGAR 

-APERTA O BOTÃO NOVAMENTE

DOIS BEEPS PARA DESLIGAR

__________________
2. DEVE SER CAPAZ DE IGNORAR TODOS OS EVENTOS NOS SENSORES ENQUANTO O ALARME ESTIVER DESLIGADO
-COM O ALARME DESLIGADO, TENTAR ATIVAR O SENSOR POR 30 SEGUNDOS

NÃO DEVE ACONTECER NADA
_________________
3. NÃO DEVE DISPARAR DENTRO DO TEMPO DE ATIVAÇÃO APÓS SER ATIVADO
- ATIVAR ALARME, E INICIAR ATIVAÇÃO DOS SENSORES

O ALARME NÃO DEVE DISPARAR ANTES DO TEMPO DE TOLERANCIA`
__________________
4. APÓS ALARMADO, E SUPERADO O TEMPO DE ACIONAMENTO, DEVE DISPARAR O ALARME APÓS O TEMPO DE CHEGADA CASO OCORRA EVENTO NOS SENSORES (EVENTO INDESEJADO) // SE TESTADO NA SEQUENCIA DO ANTERIOR IGNORAR OS DOIS PASSOS INICIAIS
//- LIGAR O ALARME
//- ESPERAR O TEMPO PARA SAIR
- ACIONAR SENSOR

DEVE EMITIR UM BIP COMPRIDO, E APOS O TEMPO DE CHEGADA DISPARAR O ALARME, TOCANDO A SIRENE E FAZENDO UMA LIGAÇÃO
__________________
5. DEVE FAZER LIGAÇÕES A CADA INTERVALO DE LIGAÇÕES ENQUANTO NENHUMA LIGAÇÃO FOR ATENDIDA // SE TESTADO NA SEQUENCIA DO ANTERIOR IGNORAR OS DOIS PASSOS INICIAIS (INICIAR CRONOMETRO JUNTO COM A SIRENE)
//- LIGAR O ALARME
//- ESPERAR O TEMPO PARA SAIR
//- ACIONAR O SENSOR
//- AGUARDAR O TEMPO AJUSTADO

COM A SIRENE DISPARANDO E APÓS A PRIMEIRA LIGAÇÃO, UMA NOVA CHAMADA DEVE SER RECEBIDA DEPOIS A CADA INTERVALO DE TEMPO DEFINIDO ENQUANTO UMA LIGAÇÃO NAO FOR ATENDIDA (DEIXAR TOCAR UMA E IR PARA O PRÓXIMO PASSO)
__________________
6. NÃO DEVE CONTINUAR FAZENDO LIGAÇÕES SE A CHAMADA FOR ATENDIDA E A SENHA INSERIDA
//- LIGAR O ALARME
//- ESPERAR O TEMPO PARA SAIR
//- ACIONAR O SENSOR
//- AGUARDAR O TEMPO AJUSTADO
-ATENDER A LIGAÇÃO

APÓS UM INTERVALO DE TEMPO NÃO SE DEVE RECEBER MAIS LIGAÇÕES
__________________
7. DEVE DESENGATILHAR APÓS O TEMPO LIMITE ENGATILHADO (AJUSTAR TEMPO PEQUENO PARA TESTE) // SE TESTADO NA SEQUENCIA DO ANTERIOR IGNORAR OS DOIS PASSOS INICIAIS
//- LIGAR O ALARME
//- ESPERAR O TEMPO PARA SAIR
//- ACIONAR O SENSOR
- AGUARDAR O TEMPO LIMITE AJUSTADO

O ALARME DEVE SER DESENGATILHADO EMITINDO 3 BIPS
__________________

8. APÓS ALARMADO, E SUPERADO O TEMPO DE ACIONAMENTO, SE IDENTIFICAR MOVIMENTO NÃO DEVE DISPARAR DENTRO DO TEMPO DE TOLERANCIA (CHEGANDO EM CASA)
- LIGAR O ALARME
- ESPERAR O TEMPO PARA SAIR
- ACIONAR O SENSOR
- DESLIGAR O SENSOR DENTRO DO TEMPO

DEVE EMITIR UM BIP COMPRIDO NO ACIONAMENTO, E 2 BIPS AO DESLIGAR;
__________________

DEVE ENCERRAR CHAMADAS AO DESENGATILHAR
- LIGAR O ALARME
- ESPERAR O TEMPO PARA SAIR
- ACIONAR O SENSOR
- AGUARDAR GATILHO E ESPERAR LIGAÇÃO
- DESENGATILHAR

A CHAMADA DEVE SER ENCERRADA.
__________________
DEVE DESENGATILHAR APÓS RECEBER SMS COM A SENHA
- LIGAR O ALARME
- ESPERAR O TEMPO PARA SAIR
- ACIONAR O SENSOR
- AGUARDAR GATILHO E ESPERAR LIGAÇÃO
- MANDAR SMS ESCRITO O STOP_CALLING_KEY

APÓS ALGUNS SEGUNDOS DEVE DESENGATILHAR
